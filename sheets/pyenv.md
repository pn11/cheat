# Pyenv

## Installation

`git clone https://github.com/pyenv/pyenv.git ~/.pyenv`

- <https://github.com/pyenv/pyenv#installation>

## Python Installation

```sh
pyenv install --list
pyenv install 3.8.0
```

## Setting version

To list versions,

```sh
pyenv versions
```

For global,

```sh
pyenv global 3.8.0
```

For local,

```
pyenv local 3.8.0
```
